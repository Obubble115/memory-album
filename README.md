# 我的图集纪念册

这是一个从零搭建的中文个人图集纪念册静态网站。

## 本地使用

直接双击打开 `index.html` 即可使用。你可以上传照片、填写标题、日期、地点、分类和文字说明。

照片和文字默认保存在当前浏览器的 `localStorage` 中，不会自动上传到互联网。

## 公开给别人看

想让别人访问，需要把网站部署到公开托管服务，例如：

- GitHub Pages：免费，适合静态网站
- Vercel：免费额度够个人网站使用，部署很方便
- Netlify：免费额度够个人网站使用，适合静态网站

域名不是必须买。没有域名也可以使用平台提供的免费网址，例如 `xxx.github.io`、`xxx.vercel.app` 或 `xxx.netlify.app`。

如果想要更正式、好记的网址，比如 `yourname.com` 或 `yourname.cn`，再购买域名即可。域名通常按年付费。

## 公开照片数据

浏览器里上传的照片只会保存在你自己的设备上，别人打开网站时不会自动看到。

如果要让所有访问者都看到同一套图集，请把照片放进 `photos/` 文件夹，然后按 `album-data.example.json` 的格式创建 `album-data.json`。部署后，网站会自动读取这个公开图集数据。
