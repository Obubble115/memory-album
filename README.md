# 寻物小占

这是一个纯静态的中文线上寻物工具，基于小六壬与梅花易数的传统思路，为用户生成失物寻找提示。

## 功能

- 失物属性选择：类别、颜色、大小
- 遗失时间选择：日期与时辰
- 场景选择：家中、办公室、路上、餐厅等
- 方位选择：八方位或不确定
- 综合输出：优先方向、可能位置、小六壬结果、梅花卦象、搜索步骤

## 部署

本项目可以直接部署到 GitHub Pages。仓库设置保持：

- Source: Deploy from a branch
- Branch: main
- Folder: / root

上传或更新 `index.html`、`styles.css`、`app.js` 后，原来的 GitHub Pages 网址会自动更新为新网站。

## 说明

当前版本没有服务器后台，所有判断逻辑都在浏览器端运行。GitHub Pages 只支持静态网站，如果以后需要隐藏算法、登录、付费、保存用户记录，需要迁移到带后端的平台。
